# Sales-Analytics
У цій роботі проведено збір та підготовку даних, які допоможуть аналізувати динаміку створення акаунтів, активність користувачів щодо листів ,  а також оцінювати поведінку в категоріях. Дані дозволять порівнювати активність між країнами, визначати ключові ринки та сегментувати користувачів за різними параметрами
