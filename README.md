У цій роботі проведено збір та підготовку даних, які допоможуть аналізувати динаміку створення акаунтів, активність користувачів щодо листів (відправлення, відкриття, переходи), а також оцінювати поведінку в категоріях, як-от інтервал відправлення, верифікація акаунтів і статус підписки. Дані дозволять порівнювати активність між країнами, визначати ключові ринки та сегментувати користувачів за різними параметрами».
